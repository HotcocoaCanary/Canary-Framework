# 安全策略

## 报告漏洞

如果你发现安全漏洞，请不要创建公开 Issue。请通过以下方式私下报告：

- 发送邮件至 Hotcocoa.Canary@outlook.com
- 或在 GitHub 上通过 [私有安全报告](https://github.com/HotcocoaCanary/Canary-Framework/security/advisories/new) 提交

## 处理流程

1. 我们将在 48 小时内确认收到报告
2. 我们将在 7 天内提供初步评估
3. 修复后我们将在 GitHub 发布安全公告
